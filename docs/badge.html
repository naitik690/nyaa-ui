Badge

A badge consists of a small circle, containing a number or filled with color. It is used for multiple purposes like to show active online, also used with notification and with e-commerce app like with cart and wishlist.

Badge Different Sizes

different sizes of badge is available, you can simple define class to change the size of the badge

  1. small (badge-s)
  2. medium (badge-m)
  3. large (badge-l)

Badge With Different Color

different color of badge is available, you can simple define class to change the color of the badge.

  1. badge-primary
  2. badge-secondary
  3. badge-accent
