Image Component

For image i have used multiple size, you can directly use class name to resize the image.

Here below example i have used "img-md" medium size for the image.

Class name and respective sizes are as follow:

  • img-xs (10rem)
  • img-sm (20rem)
  • img-md (30rem)
  • img-lg (50rem)
  • img-xl (70rem)
  • img-xxl (90rem)
